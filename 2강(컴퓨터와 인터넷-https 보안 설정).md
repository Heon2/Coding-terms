# 2강(컴퓨터와 인터넷-"https 보안 설정")

- **운영체제(OS)** : 안간이 기계와 소통할 수 있게 만드는 기반 프로그램(ex. 안드로이드, iOS, 윈도우, 리눅스(오픈소스))

- **코딩** : OS에 명령을 내리는 것

- **프로그래밍 언어** : 명령을 어떤 종류로 하는지(ex. python, C++, Java ...)

- **프로그램** : 프로그래밍 언어로 미리 짜둬서 매번 똑같이 움직이게 만들어 놓은 것

- **http** : 대화를 주고 받기위한 규칙
